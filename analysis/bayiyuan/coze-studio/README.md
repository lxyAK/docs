## 项目简介

**Coze Studio** 是源自字节跳动「扣子开发平台」的一站式 AI Agent 开发工具，现已完全开源。

**核心特性**：
- 可视化 Agent 编排
- 工作流引擎
- 知识库 RAG
- 插件系统
- 多模型支持

**技术栈**：
- 后端：Go 1.23.4 + CloudWeGo Hertz
- 前端：React + TypeScript + Rush.js
- AI 引擎：Eino (字节跳动开源)
- 架构：DDD + 微服务
