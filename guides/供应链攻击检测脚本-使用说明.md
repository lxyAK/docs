# NPM供应链攻击检测脚本 - 使用说明

## 📦 脚本文件

1. **check-axios.js** - 快速axios版本检查
2. **check-supply-chain.js** - 全面供应链攻击检测

---

## 🚀 快速开始

### 1. 快速检查axios版本
```bash
node check-axios.js
```

### 2. 全面供应链攻击检测
```bash
node check-supply-chain.js
```

---

## 🔍 检测内容

### check-axios.js
- ✅ 检查项目package.json中的axios版本
- ✅ 检测是否使用了受影响版本（1.14.1, 0.30.4）

### check-supply-chain.js
- 📦 **版本指纹检测**
  - 检查package.json中的axios版本
  - 扫描package-lock.json中的嵌套依赖
  - 检测恶意包（plain-crypto-js）

- 💻 **系统痕迹取证**
  - Linux: 检查/tmp/ld.py、crontab
  - macOS: 检查RAT持久化文件
  - 检测恶意定时任务

- 🌐 **网络行为分析**
  - 检查端口8000的可疑连接
  - 检测C2通信

- 📂 **全局依赖检查**
  - 检查全局node_modules中的恶意包

---

## ⚠️ 受影响版本
- axios@1.14.1
- axios@0.30.4

---

## 🛡️ 应急建议

如果发现受影响：
1. 立即隔离受影响环境
2. 使用 `npm ci --ignore-scripts` 重装依赖
3. 72小时内轮换所有凭证
4. 推荐使用SBOM工具生成依赖清单
