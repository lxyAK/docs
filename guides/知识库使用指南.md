# OpenClaw 知识库使用指南

## 📖 概述

OpenClaw 的知识库系统基于记忆（Memory）机制，使用纯 Markdown 文件存储，支持智能搜索。

## 🗂️ 文件结构

```
workspace-professor/
├── MEMORY.md                    # 长期记忆（每次会话加载）
└── memory/
    ├── YYYY-MM-DD.md           # 每日笔记（今天和昨天自动加载）
    ├── 项目-xxx.md             # 项目相关文档
    ├── 技术-xxx.md             # 技术笔记
    ├── 学习-xxx.md             # 学习记录
    └── ...
```

## ✍️ 如何添加知识

### 1. 长期记忆（MEMORY.md）

存放持久化的事实、偏好和决策：

```markdown
# 知识库 - 长期记忆

## 用户信息
- 姓名: 张三
- 称呼: 张总
- 时区: Asia/Shanghai

## 项目清单
- [项目A] - 描述
- [项目B] - 描述

## 技术栈
- 前端: Vue.js, React
- 后端: Node.js, Python

## 偏好设置
- 代码风格: 简洁高效
- 沟通方式: 直接了当
```

### 2. 每日笔记（memory/YYYY-MM-DD.md）

存放当天的工作内容、决策、学习：

```markdown
# 2026-04-03 每日笔记

## 今日任务
- [x] 完成功能A开发
- [ ] 编写测试用例

## 重要决策
- 决定使用方案X而不是方案Y，因为...

## 遇到的问题
- 问题描述和解决方案

## 学习内容
- 学习了XX技术，关键点是...
```

### 3. 分类文档

按主题创建专门的文档：

- `项目-八亿元.md` - 项目相关文档
- `技术-Vue.md` - Vue 技术笔记
- `学习-OpenClaw.md` - 学习记录

## 🔍 如何搜索知识

### 自然语言搜索

直接告诉我："帮我找一下关于Vue的笔记"

### 命令行搜索

```bash
openclaw memory search "关键词"
```

### 工具搜索

我会自动使用 `memory_search` 工具进行语义搜索。

## 💡 使用技巧

### 1. 让我帮你记录

直接说："记住这个：..."，我会自动写入相应文件。

### 2. 定期整理

每周花5分钟整理记忆文件，归档过时内容。

### 3. 使用标签

在文档中使用 `[标签]` 格式，便于搜索。

### 4. 模板化

创建常用模板，提高记录效率。

## 🛠️ 常用命令

```bash
# 查看记忆状态
openclaw memory status

# 搜索记忆
openclaw memory search "关键词"

# 强制重建索引
openclaw memory index --force
```

## 📝 最佳实践

1. **及时记录**: 重要信息随时记录
2. **分类清晰**: 使用前缀分类（项目-、技术-、学习-）
3. **格式统一**: 保持 Markdown 格式一致
4. **定期回顾**: 每周回顾一次记忆内容
5. **清理过时**: 及时删除或归档过时信息
