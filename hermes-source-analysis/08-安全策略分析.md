## 二、Prompt 注入防护 (`agent/prompt_builder.py`)

### 2.1 上下文文件安全扫描

在 AGENTS.md、SOUL.md、.cursorrules 等文件加载前扫描：

```python
_CONTEXT_THREAT_PATTERNS = [
    (r'ignore\s+(previous|all|above)\s+instructions', "prompt_injection"),
    (r'do\s+not\s+tell\s+the\s+user', "deception_hide"),
    (r'system\s+prompt\s+override', "sys_prompt_override"),
    (r'disregard\s+(your|all)\s+(instructions|rules)', "disregard_rules"),
    (r'act\s+as\s+if\s+you\s+have\s+no\s+restrictions', "bypass_restrictions"),
    (r'<!--[^>]*secret[^>]*-->', "html_comment_injection"),
    (r'display\s*:\s*none', "hidden_div"),
    (r'translate.*execute', "translate_execute"),
    (r'curl.*\$API_KEY', "exfil_curl"),
    (r'cat\s+.*\.env', "read_secrets"),
]
```

### 2.2 不可见字符检测

```python
_CONTEXT_INVISIBLE_CHARS = {
    '\u200b',  # 零宽空格
    '\u200c',  # 零宽非连接符
    '\u200d',  # 零宽连接符
    '\u2060',  # 字连接符
    '\ufeff',  # BOM
    '\u202a',  # 从左到右嵌入
    '\u202b',  # 从右到左嵌入
    '\u202c',  # 方向格式化关闭
    '\u202d',  # 从左到右覆盖
    '\u202e',  # 从右到左覆盖
}
```

### 2.3 处理策略

检测到威胁 → **替换为警告文本**，不加载原内容：

```python
return f"[BLOCKED: {filename} contained potential prompt injection ({findings}). Content not loaded.]"
```

## 三、命令审批系统 (`tools/approval.py`)

### 3.1 审批模式

| 模式 | 行为 |
|------|------|
| `allow` | 自动批准所有命令 |
| `ask` | 危险命令需确认 |
| `deny` | 禁止终端命令 |

### 3.2 命令分类

```python
# 危险命令（需要审批）
DANGEROUS_PATTERNS = [
    r"rm\s+-rf", r"sudo\s+", r"chmod\s+777",
    r"curl\s+.*\|.*sh", r"wget\s+.*\|.*sh",
    r"mkfs", r"dd\s+if=", r">/dev/sd",
    r"systemctl\s+(stop|disable)", r"iptables\s+-F",
]
```

### 3.3 允许列表

用户可配置命令白名单：

```yaml
# config.yaml
security:
  approval:
    mode: ask
    allowlist:
      - "ls *"
      - "cat *"
      - "git *"
      - "python *"
```

## 四、路径安全 (`tools/path_security.py`)

### 4.1 路径遍历防护

```python
def validate_path(path, allowed_dirs):
    """检查路径是否在允许目录内"""
    real_path = os.path.realpath(path)
    for allowed in allowed_dirs:
        if real_path.startswith(os.path.realpath(allowed)):
            return True
    return False
```

### 4.2 工作目录限制

Agent 的文件操作限制在配置的工作目录内：
- 读取：默认允许任意路径
- 写入：限制在工作目录
- 删除：严格限制 + 二次确认

## 六、Skills 安全 (`tools/skills_guard.py`)

### 6.1 安装前扫描

从 Skills Hub 安装技能前扫描：

```python
def scan_skill(skill_dir):
    """扫描技能目录"""
    findings = []
    
    # 检查恶意代码
    for file in skill_dir.rglob("*"):
        content = file.read_text()
        if re.search(r"os\.system|subprocess|eval\(|exec\(", content):
            findings.append(("dangerous_code", file))
    
    # 检查网络请求
    if re.search(r"requests\.(get|post)|urllib|http\.client", content):
        findings.append(("network_access", file))
    
    return findings
```
