### 1.2 调度器 (`cron/scheduler.py`)

```python
def tick():
    """每 60 秒调用一次，检查并执行到期作业"""
    
    # 1. 文件锁防止并发 tick
    acquire_lock("~/.hermes/cron/.tick.lock")
    
    # 2. 获取到期作业
    due_jobs = get_due_jobs()
    
    # 3. 逐个执行
    for job in due_jobs:
        result = run_agent_for_cron(job)
        
        # 4. 静默标记检查
        if result.startswith("[SILENT]"):
            save_output_only(result)  # 仅本地保存
        else:
            deliver_to_platform(result)  # 投递到平台
    
    # 5. 推进 next_run
    advance_next_run(job)
```

### 1.3 作业存储

```

├── jobs.json          # 作业定义
├── outputs/           # 执行输出
└── .tick.lock         # 并发锁
```

## 二、委托子 Agent (`tools/delegate_tool.py`)

### 2.1 架构

```
Parent Agent (主Agent)
    ├── delegate_task(goal="任务A", context="...")
    │   └── Child Agent 1 (隔离上下文)
    │       ├── 独立对话历史
    │       ├── 独立终端会话
    │       └── 受限工具集
    ├── delegate_task(goal="任务B", ...)
    │   └── Child Agent 2
    └── 并行等待所有子Agent完成
        └── 汇总结果返回
```

### 2.2 安全边界

```python
DELEGATE_BLOCKED_TOOLS = frozenset([
    "delegate_task",   # 禁止递归委托
    "clarify",         # 禁止向用户提问
    "memory",          # 禁止写入共享记忆
    "send_message",    # 禁止跨平台发送
    "execute_code",    # 应逐步推理
])

MAX_DEPTH = 2  # 最大嵌套深度
```

### 2.3 并发控制

```python
# 默认最多 3 个并发子Agent
_DEFAULT_MAX_CONCURRENT_CHILDREN = 3

# 可配置
# config.yaml
delegation:
  max_concurrent_children: 5
```

### 2.4 可用工具集

子Agent 可访问的工具集（排除调试、安全、平台特有集）：

_EXCLUDED_TOOLSET_NAMES = {

    ├── 并行启动 Agent 实例

    ├── 可选：轨迹压缩
